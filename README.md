# Bible Quiz Game

A faith-based Java quiz game with NPC opponents, difficulty levels, and Bible trivia.

## Features
- 10 rounds with increasing point values
- Difficulty selection (Easy, Medium, Hard)
- NPC opponents: Moses, Noah, Adam
- Encouraging verses on wrong answers
- Humility verse and congrats if you win
- Replay option

## How to Run
1. Clone the repo:
git clone https://github.com/RobBed98/Semester-Project-Bible-Quiz-Game.git

2. Compile:
javac src/BibleQuizGame.java

3. Run:
java BibleQuizGame
